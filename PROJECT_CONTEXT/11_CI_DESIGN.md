## Leitsatz & Werte

**„Vertrauen ist Evidenz — nie behauptet."** Belastbar · Industriell · Klar · Ehrlich.
Substanz vor Effekt, Hierarchie vor Dekoration. Das Design ist die visuelle Form desselben
Prinzips, das auch den Code regiert (Ehrlichkeit vor Optik).

## Farben (verbindlich — auswendig können)

- **Klarwerk-Orange `#ED7D0E`** — die EINZIGE Markenfarbe; nur für Aktionen/Highlights,
  nie für Fließtext auf Weiß.
- **Stahl-Dunkel `#16222C`** — Text auf Weiß, Kontrastflächen.
- Neutrale: Text `#1B1E21` · Text 2 `#687078` · Rahmen `#E4E7EA` · Fläche `#F3F4F6` · Weiß.
- **Semantik (nie Dekoration):** Gesichert `#3AA06A` · Belastbar `#C8861A` · Konflikt `#C0473F` ·
  **KI/Reasoner `#5B50C4`** — KI-Inhalte IMMER violett kennzeichnen.
- Richtwert-Verhältnis: ~62 % Weiß/Neutral · ~24 % Stahl · ~14 % Orange.

## Typografie & UI

**IBM Plex Sans** (Text/Überschriften) + **IBM Plex Mono** (Labels, Eyebrows, Kennzahlen, IDs,
Statusmarker). Line-Icons (~1,8 Strich, runde Enden), keine Emoji als Icons. Cards Radius
12–14 px, Rahmen `#E4E7EA`. Buttons: Primär Orange · Sekundär Stahl · Tertiär Outline.
Status-Pills: Validiert / In Prüfung / Konflikt / ✦ Reasoner.

## Logo

Zielscheibe (Ring + Punkt) auf heller Kachel + Wortmarke KLARWERK (Versalien, gesperrt),
Untertitel „Reasoning System". Nie drehen/verzerren/umfärben; Mindestgröße 24 px digital.

## Bildwelt & Video (Kurzform)

Dokumentarisch: echte Menschen bei konkreter Tätigkeit, ruhige Farbwelt, Orange nur als Akzent.
**Verboten:** KI-Roboter, blauer Tech-Glow, Stockfoto-Posen, Hype. Video: ruhige Schnitte,
**Untertitel Pflicht (DE/EN)**, End-Card = Zielscheibe + Claim, KI-Inhalte violett kennzeichnen.
Praxis-Lehren (02.07.): Veo nie echte Produkt-UI generieren lassen (echten Screencast verwenden);
pro Gemini-Auftrag nur EINE Szene.

## Sprache

Nüchtern, aktiv, ohne Superlative, echte Umlaute. Kernbegriffe konsistent: Knowledge Object ·
Reasoning System · Trust/Reifegrad · Validierung · Quelle & Kontext · Erfahrungswissen · Evidenz.
Markenversprechen: „KLARWERK sichert das Erfahrungswissen, das sonst mit den Menschen geht —
als validierten, wiederverwendbaren Wissensbaustein."
